
# Dataset Backup Repository

This repository is used for storing and backing up various bioinformatics-related datasets. The data are mainly for backup purposes, and some are sourced from public databases or third-party projects.

If you have any copyright or license concerns, please contact us for removal.

## Available datasets

- `motifs/`: TF motif resources for regulatory-network inference, including
  JASPAR2024 CORE vertebrate PFMs, HOCOMOCO v14 CORE motif files and
  annotations, and human motif objects.
